# 3D Character Assets

Replace placeholder SVGs with pre-rendered PNG/WebP (transparent background).

| File | Usage |
|------|--------|
| `baby-3d.png` | Младенец — infant, deficit, product baby |
| `child-immunity-3d.png` | Ребёнок — innate/adaptive immunity |
| `schoolboy-3d.png` | Школьник — risk factors, dose |
| `student-brain-3d.png` | Школьник — cognitive |
| `sleep-child-3d.png` | Ребёнок — mood & sleep |

Until PNGs are added, placeholders from `placeholders/` are shown automatically.
