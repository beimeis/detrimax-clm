# Детримакс® — CLM-презентация

Интерактивная CLM-презентация для медицинских представителей по бренду Детримакс®.

## Формат

- Вертикальный планшетный экран **1024×1366**
- Без скролла, адаптивное масштабирование
- React + Vite + TypeScript

## Маршруты

1. **Младенец 0–12 месяцев** — рост, дефицит, иммунитет, продукт Бэби
2. **Иммунитет и частые ОРВИ** — защита, адаптивный иммунитет, продукт Актив
3. **Школьник** — факторы риска, когнитивное развитие, сон, продукт Актив

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```
